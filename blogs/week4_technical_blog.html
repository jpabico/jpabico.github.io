Ruby's Enumerable#group_by Method

April 4, 2014

The Ruby group_by method is a useful method for separating data into groups based on the results of passing that data into a block of code. It is applied to an enumerable. The result is a hash where the keys are the evaluated result from the block and the values are arrays of elements in the collection that correspond to the key. Its general for is:

Enumerable.group_by { |object| block of code} -> returns a hash
Here are some examples of how #group_by is used. There are many more ways to apply the group_by method than listed below. Try experimenting yourself!
  • Example 1 - an array of numbers is separated into two groups: even (.even? == true) and odd (.even == false)
    [3, 6, 7, 21, 34, 8, 0, 5].group_by {|x| x.even?}
        #=> {false=>[3, 7, 21, 5], true=>[6, 34, 8, 0]}
  • Example 2 - each number in a range of numbers is separated according to the remaineder when divided by 4
    (1..10).group_by { |i| i%4 }
        #=> {0=>[4, 8], 1=>[1, 5, 9], 2=>[2, 6, 10], 3=>[3, 7]}
  • Example 3 - the strings in the array are ordered by first letter
    fruits = ["apples", "grapes", "apricots", "mangoes", "bananas"] group_by_first_letter = fruits.group_by { |x| x[0] }
        #=> {"a"=>["apples", "apricots"], "g"=>["grapes"], "m"=>["mangoes"], "b"=>["bananas"]}
