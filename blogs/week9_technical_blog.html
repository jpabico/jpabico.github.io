A Beginner's Guide to Regular Expressions

May 4, 2014

A regular expression is a sequence of characters that forms a search pattern. Each character in a regular expression represents a literal meaning or a variety of special meanings. Here, I will show some regular expressions and their meaning so you can learn by example. All regular expressions are written between two forward slashes / -regular expression goes here- /. A caret ^ indicates the beginning of the expression. A dollar sign $ indicates the end of the expression.

Matching a User Name

/^[a-z0-9_-]{3,16}$/
  • ^ beginning of expression
  • [a-z0-9_-] any lowercase letter (a-z), number (0-9), an underscore, or a hyphen
  • { 3,16} at least 3 character but no more than 16
  • & end of expression

At the core of jQuery is the ability to select some elements and do something with them. The basic syntax is: 
$(selector).action()
Here are some example of the meaning of the syntax: 
$('p')   // select all the < p > elements in the document
$('#profile')   // select html elements that have an id of "profile"
$('.important')   // select html elements that have a class of "important"
$('li.important')   // select all list items with class "important"
$('#profile input')   // select all < input > elements that are descendants of an element with the id "profile"
All of the above return array-like objects (i.e. there is a ".length" method, array[index] can be used to access particular elements of the return, etc)

Once a selection has been chosen, methods can be called on the selection. Here are some example of the meaning of the syntax: 

$('li').text()   // get the text content of all < li > elements
$('li').text("Hello")   // set the text content of all < li > elements to "Hello"
$('input').val()   // get the value of the first < input > element
$('input').val('')   // set the value of the first < input > element to an empty string

//Note above that jQuery uses the same method name for getting and setting

$('#picture').hide()   // hide HTML elements that have an id "picture"
$('#picture').show()   // show HTML elements that have an id "picture"
